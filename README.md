# Line启动器

`Line启动器`的主仓库。下文简称*本启动器/启动器/Line/LMC/*。本仓库仅用于简要介绍LMC，不包含任何代码。

## 简介
`Line启动器(Line Launcher)` 是一款为`Java`版[Minecraft](https://minecraft.net)所打造的游戏启动器，能够进行微软登录、游戏的下载安装等功能。  
它有两个版本，一个由`C#`开发，主开发者是[皇鱼](https://github.com/tmdakm)，这是[代码仓库](https://github.com/LinearTeam/LineLauncherCs/)。

另一个版本是`Visual Basic`版本。由[影宝](https://github.com/iloveei)开发，这是[代码仓库](https://github.com/LinearTeam/LineLauncherVb)。

二者都基于[`Avalonia`](https://github.com/AvaloniaUI/Avalonia)开发，支持跨平台。
